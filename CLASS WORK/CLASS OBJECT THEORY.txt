		syntax:
					class <class_name>
					{
						access specifiers:
						 member function
						 data members						
					};
			
	Access Modifier : 
	
	They help the members to show their visibility.
			
					
	There are 3 Access Specifier/Modifier
	1) public:	
			The member is visible within and outside the class.			
	2) private:
			The member is visible within its class only.	
	3) protected:
			The member is visible within the class and 
			also to its sub classes.	
			
	By default the access specifier in class is private.
